# Star Wars Guide (Javascript)

A Star Wars Character Guide built with JS (including Async functions and JSON)

## Prerequisites

None

## Getting Started

On terminal:
1. Run `git clone https://github.com/sgtbrunner/star-wars-guide-js.git`
2. Run `cd star-wars-guide-js`
3. Open `index.html`

## Built With

* HTML
* CSS
* Javascript
* Restful API
